整理今日思路
学习了具体git日常工作流程
先找到自己内存满满的盘符 创建属于自己的工作文件夹 右键git here进入工作状态
git clone 公司git网址 // 克隆先辈资料项目(克隆一次，后续都可以进行拉取得到)
git status //日常检查
git add . //日常存入本地
git checkout -b 喜欢的名字 //创建自己的分支 修改自己的放心
现在已经切换过来了
然后日常操作    检查 存本地 git commit -m "今日份beg"提交成功 检查 然后切换主分支 连接关联仓库
git remote add origin git@github.com:tddch/abab.git
然后上传把本地仓库的项目提交到远程仓库
git push -u origin master
看到100%
OK第一天完成
